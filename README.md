# CSE 163 Final Project – Meal Plan Analysis

## Requirements
Python 3.9+
pandas
matplotlib

Install dependencies:

pip install pandas matplotlib

## Files

mealplan_analysis.py
Main script that loads data, cleans datasets, and generates visualizations.

test_mealplan.py
Testing file with assertions verifying cleaning functions.

food_expenditure.csv
USDA food expenditure dataset.

student_food_survey.csv
Student survey dataset.

## Running the Project

Run the main analysis script:

python mealplan_analysis.py

This will generate figures inside the /figures directory.
